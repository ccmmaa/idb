MusePy API


api.musepy.me/song

Returns all Songs

api.musepy.me/song/songID

Returns one Song

api.musepy.me/artist

Returns all Artists

api.musepy.me/artist/artistID

Returns one Artist

api.musepy.me/album

Returns all Albums

api.musepy.me/album/albumID

Returns one Albums

api.musepy.me/city

Returns all Cities

api.musepy.me/city/cityID

Returns one City
